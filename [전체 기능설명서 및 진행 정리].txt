단계	내용	비고

🏫 당신이 기획한 전체 기능 정리 (구체적 정리)
1. 로그인
비밀번호 입력 후 로그인
권한별 메뉴 자동 제어

2. 권한체계
원장, 실장 → 모든 기능 + 전체 학생 접근
팀장 → 학생관리, 학생정보, 시험입력, 시간표출력
강사 → 시험입력, 시간표출력
조교 → 학생관리, 시험입력, 시간표출력

팀장,강사,조교 → 학생접근은 전체or담당을 원장과실장이 선택하여 부여할 수 있도록 설계

3. 학생관리
등록 : 학생 1명씩 입력 or 엑셀 업로드

수정 : 추후 개발 예정

삭제 : 선택삭제/전체삭제 (삭제 확인창 필수)

엑셀양식 다운로드 가능

UI : 박스 스타일 카드, 가로정렬, 저장버튼

4. 시험입력
시험일정 등록

과목 다중선택

시험정보 저장

(추가예정) 시험정보 전체삭제 기능 (권한제어)

깔끔한 고급 스타일 적용

5. 시간표출력 (앞으로 작업 예정)
학생별, 강사별 시간표 출력 기능

6. 현황보고 (추후 작업)
전체 학생수, 반별 학생수 등 자동 통계
